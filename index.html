HTML

Tag/ Attribute Description Notes
&lt; and &gt; Displays < and >, respectively.  
&nbsp; Allows for empty table cells without breaking formatting.  
&amp; Displays the character & without the browser assuming it's code.  
<a></a> Anchor tag. For linking to other sites or within the page. href="link" used inside the opening tag, while name of the hyperlink in between.
In order to link using an image, the above applies, but the image is linked using <img src="link"> followed by text, if desired.
In order to link to any section of the same page, href=#id is used inside the opening tag.
target="_blank" can be used inside the opening tag in order to open the link in a new tab.
<br> Line break. Creates a paragraph.
<!-- text --> For creating a comment within the editor.  
<ol></ol> Ordered list.  
<ul></ul> Unordered list.  
<li></li> Individual list elements used within ordered and unordered tags.  
<em></em> Italics. Inline.
<strong></strong> Bold. Inline.
<table></table> Main table container. Semantical (Probably lmao).
<tbody></tbody> Table body. Parent to <tr> Semantical.
<thead></thead> Table heading. Semantical.
<tfoot></tfoot> Table footing. Semantical.
<tr></tr> Starts a new table row.  
<th></th> Table heading cells. Subordinate to <tr>
<td></td> Individual table cells pertaining to each row.  
rowspan="1" Attribute of <td>. Collapses a cell vertically by however much the value is. Default value of 1.
colspan="1" Attribute of <td>. Collapses a cell horizontally by however much the value is Default value of 1.
<style></style> Allows to apply CSS code within the html file. Only useful if the website has only one page (thus 1 html file). Must be included within the <head> of the document.

Helpful notes
More than one stylesheet can be imported by using a separate link element for each.

CSS

Selector/ Attribute Description Notes
* {} Universal selector. Targets all elements of the page.  
#id {} Id selector. Targets the elements with that particular id name.  
li>a {} Child selector. In this example, all <a> elements which are children of <li> elements are targeted.  
p a {} Space is used as a descendant selector. <a> elements nested within <p> elements are targeted. Broader than >, above.
h3.class {} Selects elements which are both h3 and .class.  
h1+p {} Adjacent sibling selector. Targets the first, and only the first, <p> after an <h1> element.  
h1-p {} General sibling selector. Targets all <p> siblings of <h1> elements.  
inherit; Used after any property to match the same property value of the parent element.  
min-width: Minimum width.  
max-width: Maximum width.  
min-height: Minimum height.  
max-height: Maximum height.  
overflow: Dictates how content of the box should behave if it doesn't fit in the browser window. Can assume values hidden or scroll.
border-width: Four values clockwise from top. Values can be stated in px or thin, medium, thick.
border-top-width: Used in order to define the thickness of any one side of the box individually.  
border-color: Four values as above.  
border-top-color:    
border-style: Can take up the following values:
solid A single solid line.
dotted Squares with the same width as the border.
dashed Series of short lines.
double Two solid lines. The total thickness mediated by the border-width value.
groove Appears to be carved into the page.
ridge Appears to stick out from the page.
inset Appears embedded into the page.
outset Looks like it is coming out of the screen.
hidden / none No border shown.
border: Shorthand way of defining border width, style and color at once.  
border-top-style: Used to individually change border styles.  
padding-top: Defining padding of individual borders of boxes using px, etc.  
padding: Shorthand form. Can take four clockwise values or two (sides and top-bottom).  
margin-top: Defining margins of individual borders of boxes using px, etc. Margins are collapsed, i.e. the margin with the highest value sets the margin betweent two boxes.
Setting the left and right margins to auto centers the box within the page.
margin: Shorthand form. Can take four clockwise values or two (sides and top-bottom).
display: This property can take four values:
  • inline causes a block level element to behave as inline.
  • block causes an inline level element to behave as block.
  • inline-block causes a block level element to behave as inline while retaining the properties of a block element.
  • none hides the element. In this case, the box is visible only in the source code but invisible in the browser.
  
visibility: hidden This is used in order to hide an element but leave the space that it would otherwise occupy. visible is the default value.
border-image: url("...") This property allows to set an image as the border of a box.
Four values are required after the link to set the width of each border, followed by one of three values:
  • stretch stretches the image accross the lenght of the border.
  • repeat repeats the image accross the lenght of the border.
  • round similar to repeat but scales the tile image for a perfect fit.
 Example in use: border-image: url("...") 10 9 10 9 round;.
-moz-border-image: and -webkit-border-image: can be used for browser compatibility.
box-shadow: Creates a shadow around the box. Takes in four numerical values, positive or negative, followed by the color of the shadow.
The numerical values determine, in order: horizontal offset, vertical offset, blur distance and the spread of the shadow.
inset can be used before the numerical values in order to create an inner-shadow.		 -moz-border-image: and -webkit-border-image can be used for browser compatibility.
border-radius: Rounds the corners of the border. In order to target individual corners, the format border-top-right-radius: can be used.
Shorthand form is accepted. In order to create an eliptical shape the following shorthand could be used 10 10 10 10 / 5 5 5 5; where 10 defines the horizontal values and 5 the vertical.
A circle shape can be created by setting the corner values equal to the height of the box.		  
position: Accepts the following values:
  • static normal flow, the default value.
  • relative flow relative to what it would've been by default.
  • absolute takes the box out of normal flow, other boxes ignore its positioning.
  • fixed positions the box relative to the browser window.
 Offset values are used in tandem in order to place the box where needed. Those take the form of top, right, left and bottom along with general values.
z-index: Used in situations where boxes can overlap. Takes in whole numbers, where a higher value places the box higher up. When boxes overlap, the default behaviour is for the later html code be placed higher up.
float: This property is used in order to place the box as far to the left or right as possible. Other elements flow around it. A width should be specified as well.
Possible values are right and left.		 On some browsers, in cases where a parent element contains only floated elements, it can collapse, ruining the layout. In order to fix this the following parameters can be set overflow: auto; and width: 100%;.
clear: This property is used in order to control the way boxes stack within the same containing element.
The possible values it can take are:
  • left the left-hand side of the box should not touch any other elements.
  • right the right-hand side of the box should not touch any other elements.
  • both neither side of the box should touch another element.
  • none both sides can touch other elements.
  
font-size: Sets size of the font.  
background-color: Sets the background color for the element.  
color: Sets the foreground color. Or text color if that's the case.  
opacity: Sets the opacity, from 0 to 1.  
font-weight: Selects how thick the font should be. Values can be either normal or bold, or numerical values ranging from 100-900 in multiples of 100. The default value is 400.  
text-align: Sets the text alignment. Values can be left, center or right.  
background-image: Sets a background image using url("...").  
font-family: Selects the font type. In the case of a multi-word font name, "" should be used. Fonts can be separated by a comma, as fallback options.  
font-style: italic; Used to set a font to italic.  
@font-face {
font-family: "name of the font";
src: url("...") format("truetype");
}		 Used in order to import a font to be used later on. Fonts can also be linked in html. The code should sit in the head section and is as follows:
<link href="..." rel="stylesheet">.
line-height: Determines the spacing between lines of text. Values can be in px or whole numbers signifying the ratio against the height of the font. line-height: 3; would result in a line spacing 3 times as large as the font height.

Helpful notes
More than one stylesheet can be imported by using @import url("...") before any other rules.
When specifying colors in RGB values, a fourth number can be used which determines opacity, from 0-1.

Other notes

In linking to path files, ./ at the start of the url points to the current location within the folder.
